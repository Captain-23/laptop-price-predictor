
# Laptop Price Predictor

An interactive web app that predicts laptop prices using machine learning.

## Features
- Real-time price prediction
- Dynamic dropdowns by brand/model
- Interactive Plotly price comparison
- Similar laptop recommendations
- Modern UI built with Streamlit

## Tech Stack
- Python, Pandas, Scikit-learn
- Streamlit (Frontend)
- Plotly (Visualization)
- Random Forest Model
